2026 Championship chase

How the points work

Points are tracked by class all season for Late Models, A-Modifieds, B-Modifieds and Hornets, and updated after each race night. Drivers earn points by class for showing up and finishing, with the championship decided at the end of the season. Standings run live on MyRacePass, so the numbers you see there are the official ones.
